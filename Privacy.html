Resumen claro

Compromiso de privacidad

Purple Jack utiliza solo la información necesaria para operar sus funciones dentro de Discord. El bot no solicita contraseñas, tokens, correos electrónicos, direcciones IP, datos de pago ni información financiera real. Los datos tratados se limitan a identificadores de Discord y registros internos necesarios para la economía ficticia, juegos, cooldowns, roles temporales, inventario y administración funcional del bot.

01

Identidad del bot y alcance

Purple Jack es un bot de Discord desarrollado para uso exclusivo dentro de SuperSus SA Oficial. Su finalidad es ofrecer mecánicas gaming internas de entretenimiento, economía ficticia, tienda de artículos, inventario, rankings, minijuegos, roles temporales y herramientas de operación comunitaria.

Purple Jack no es un servicio público general, no está afiliado oficialmente con Discord Inc. y no representa a Discord. Discord es la plataforma sobre la cual opera el bot mediante sus API oficiales.

02

Datos que Purple Jack puede tratar

El bot procesa datos mínimos y funcionales. Estos datos permiten identificar usuarios dentro del sistema, mantener progreso, aplicar límites de uso y ejecutar las mecánicas internas del bot.

Dato Descripción Finalidad
ID de usuario de Discord Identificador numérico proporcionado por Discord. Asociar balance, inventario, cooldowns y progreso al usuario correcto.
ID de servidor, canal o rol Identificadores técnicos de Discord usados por el bot. Limitar el funcionamiento al entorno autorizado, publicar respuestas y gestionar roles temporales.
Balance, banco e inventario Registros internos de moneda ficticia, artículos virtuales y cantidades. Operar la economía ficticia, tienda, compras, recompensas y comandos de consulta.
Cooldowns y marcas de tiempo Fechas u horas de uso de comandos y juegos. Prevenir abuso, mantener equilibrio y controlar tiempos de espera.
Registros operativos Eventos como compras, uso de items, ajustes staff o acciones económicas relevantes. Auditoria interna, soporte, seguridad y control operativo del bot.
Roles temporales ID de usuario, ID de servidor, ID de rol y tiempo de expiración. Asignar y retirar roles temporales comprados o activados mediante el bot.

Datos que Purple Jack no solicita ni almacena

  • Contraseñas, tokens de acceso o credenciales de Discord.
  • Datos bancarios, tarjetas, cuentas de pago, direcciones o documentos personales.
  • Correos electrónicos, números de teléfono o direcciones IP.
  • Contenido privado fuera de las interacciones necesarias con el bot.
  • Datos sensibles no necesarios para la funcionalidad declarada.
03

Economía ficticia y ausencia de apuestas reales

El sistema económico de Purple Jack es totalmente ficticio. Las monedas, balances, banco, articulos, recompensas, rankings, juegos y resultados internos no representan dinero real, no tienen valor monetario fuera del servidor y no pueden convertirse, retirarse, venderse ni intercambiarse por divisas reales.

Purple Jack no procesa pagos reales, apuestas con dinero real, premios monetarios, retiros, depositos, conversiones financieras ni transacciones externas. Sus juegos y mecánicas de economía son recreativos y están limitados al entorno comunitario de SuperSus SA Oficial.

Cualquier referencia a monedas, banco, tienda, recompensas o juegos dentro del bot debe entenderse como parte de una simulación interna de entretenimiento sin relación con servicios financieros, azar regulado o apuestas con divisa real.

04

Cómo se utilizan los datos

Los datos se utilizan exclusivamente para prestar las funciones declaradas del bot dentro de su entorno autorizado. Purple Jack no vende, alquila ni comercializa datos de usuarios.

  • Economía: consultar y actualizar balance, banco, rankings y recompensas ficticias.
  • Tienda e inventario: gestionar compras, stock, artículos virtuales y uso de items.
  • Juegos recreativos: aplicar resultados, límites, cooldowns y balances ficticios.
  • Roles: asignar o retirar roles temporales cuando una mecánica autorizada lo requiera.
  • Administracion: permitir acciones staff, auditoria operativa y soporte funcional del bot.
  • Prevencion de abuso: limitar spam de comandos, duplicaciones, uso excesivo o acciones inconsistentes.
05

Cumplimiento con Discord Developer

Purple Jack procura operar de forma compatible con las reglas aplicables a desarrolladores de Discord, incluyendo el principio de utilizar datos de API solo para la funcionalidad declarada y necesaria del bot.

  • No se solicitan credenciales, tokens ni acceso externo a cuentas de Discord.
  • No se usan datos de Discord para publicidad, venta de datos, perfilado comercial o discriminación.
  • No se contacta a usuarios fuera de Discord usando datos obtenidos mediante el bot.
  • No se realiza scraping, minería masiva ni extracción de información ajena a la funcionalidad del bot.
  • No se utiliza contenido de mensajes para entrenar modelos de inteligencia artificial.

Para información de plataforma, Discord mantiene sus propias políticas y términos oficiales. Esta política describe únicamente el tratamiento de datos realizado por Purple Jack dentro de su operación autorizada.

06

Almacenamiento, seguridad y acceso

Purple Jack almacena la información funcional en una base de datos y utiliza caché en memoria para mejorar la velocidad de respuesta del bot. El acceso a los datos se limita al sistema del bot y al responsable o personal técnico autorizado que necesite intervenir por mantenimiento, soporte o auditoría.

  • Los accesos técnicos se mantienen restringidos al responsable y equipo autorizado.
  • Las credenciales sensibles deben permanecer fuera del código fuente y gestionarse como configuración segura.
  • La caché en memoria reduce latencia y se sincroniza periódicamente con la base de datos.
  • Los logs operativos se usan para trazabilidad, soporte y control interno del bot.
Capa Uso Persistencia
Cache en memoria Respuesta rapida y operaciones frecuentes. Temporal; se limpia o sincroniza periodicamente.
Base de datos Conservación de balance, inventario, cooldowns y configuraciones. Persistente mientras el bot opere y el dato sea necesario.
Canales de log internos Registro de eventos relevantes de economía, tienda o staff. Segun configuracion y necesidades operativas del bot.
07

Servicios de terceros y divulgación

Purple Jack opera sobre Discord y puede depender de infraestructura técnica para alojamiento y base de datos. Fuera de lo necesario para que el bot funcione, los datos no se comparten con terceros, no se venden y no se entregan a redes publicitarias o intermediarios de datos.

  • Discord: plataforma donde ocurre la interacción y proveedor de los identificadores técnicos.
  • Proveedor de alojamiento: ejecuta la aplicación del bot.
  • Proveedor de base de datos: almacena la información funcional necesaria.
08

Retención y eliminación de datos

Los datos se conservan mientras sean necesarios para operar Purple Jack dentro de SuperSus SA Oficial, mantener la economía ficticia, resolver incidencias o cumplir funciones administrativas del bot.

  • Los usuarios activos conservan su progreso, balances, inventario y cooldowns.
  • Los datos inactivos pueden ser limpiados o archivados a criterio del responsable.
  • Si un usuario solicita eliminación, el responsable revisará la solicitud y eliminará o anonimizará los datos cuando sea razonablemente posible.
  • Al retirar el bot del servidor, los datos asociados podrán conservarse durante un periodo razonable para respaldo, auditoría o transición.
09

Derechos y solicitudes de usuarios

Los usuarios pueden solicitar información sobre los datos funcionales asociados a su ID de Discord dentro de Purple Jack. También pueden solicitar corrección, revisión o eliminación de datos cuando corresponda.

  • Acceso: solicitar un resumen de balance, inventario, cooldowns u otros datos funcionales.
  • Corrección: reportar errores o inconsistencias para revisión.
  • Eliminación: pedir la eliminación de datos asociados al usuario, cuando sea compatible con la seguridad y auditoría del bot.
  • Soporte: reportar fallos, abusos o dudas sobre el funcionamiento del bot.
10

Privacidad de menores

Purple Jack no está dirigido a menores de 13 años ni a usuarios que no cumplan la edad mínima requerida por Discord en su jurisdicción. El bot opera dentro de Discord y presume que los usuarios cumplen los requisitos de edad de la plataforma.

Si se detecta un caso que requiera revisión, el responsable podrá tomar medidas de moderación, restricción o eliminación de datos según corresponda.

11

Cambios en estas políticas

Estas políticas pueden actualizarse para reflejar mejoras técnicas, cambios de funciones, ajustes de seguridad o requisitos de Discord Developer. Cuando se realicen cambios relevantes, se actualizará la fecha de este documento y, si corresponde, se comunicará dentro del entorno oficial de SuperSus SA Oficial.
