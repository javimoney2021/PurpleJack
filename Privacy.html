PurpleJack Bot

Privacy Policy

Effective Date: June 11, 2026
This Privacy Policy describes how PurpleJack Bot collects, uses, and retains data when you interact with it on Discord. We are committed to transparency and handling your data responsibly and minimally.
01

Who We Are

PurpleJack Bot ("the Bot", "we", "our") is an independent Discord bot providing a virtual economy experience. The Bot operates exclusively within a single authorized Discord server and is not affiliated with or endorsed by Discord Inc.

For privacy inquiries, please contact the server administration through the official authorized Discord server.

02

Data We Collect

The Bot collects only the data strictly necessary to provide its economy and game features. No personal information beyond Discord-provided identifiers is ever collected.

Data Type Description Purpose
Discord User ID A unique numeric identifier provided by Discord Identify user accounts in the economy database
Discord Guild ID Numeric identifier of the Discord server Restrict Bot operation to authorized servers only
Virtual Balance In-Bot wallet and bank amounts (virtual currency) Economy system functionality
Cooldown Timestamps Unix timestamps of last command usage Enforce fair-use cooldowns on commands
Employment Data Current job, hire date, work history, XP, earnings Employment simulation system
Inventory Data Virtual items owned and their usage dates Virtual shop and item system
Game History Records of game sessions and outcomes Leaderboards and statistics features
Role Assignments Discord role IDs for temporary roles granted by the Bot Temporary role management system
We do not collect: real names, email addresses, IP addresses, payment information, message content, voice data, or any sensitive personal information.
03

How We Collect Data

Data is collected exclusively through direct interaction with the Bot's commands and features within Discord. Specifically:

  • When a user executes a Bot command (!bal, !trabajar, etc.), their Discord User ID is recorded automatically
  • Economy values are updated as a direct result of game outcomes and commands
  • No passive data collection occurs — the Bot does not monitor messages, reactions, or activity outside of explicit Bot command interactions
04

How We Use the Data

All collected data is used solely to operate the Bot's features. Specific uses include:

  • Economy functionality: tracking virtual balances, transactions, and cooldowns
  • Employment system: saving job progress, XP, and work history
  • Fair use enforcement: applying cooldowns to prevent command abuse
  • Leaderboards: displaying anonymized rankings within the server
  • Administrative tools: allowing server staff to manage the economy

Data is never used for advertising, profiling, or any commercial purpose.

05

Data Storage and Security

Data is stored in a PostgreSQL database hosted on a secure cloud infrastructure. The following security measures are in place:

  • Database credentials are stored as environment variables and never hardcoded
  • Database access is restricted to the Bot's application only
  • An in-memory cache layer (RAM) is used to minimize direct database access and reduce latency
  • Cache is periodically flushed to the persistent database to maintain consistency
  • No data is stored locally on users' devices
Storage Layer Type Persistence
RAM Cache In-memory (temporary) Session only; flushed every 5 minutes
PostgreSQL Database Relational database Persistent long-term storage
06

Data Sharing and Third Parties

We do not sell, rent, share, or disclose any user data to third parties under any circumstances, with the following limited exceptions:

Party Data Shared Reason User Control
Discord Inc. Bot interactions (standard API usage) Required to operate on Discord's platform No opt-out
Cloud Host (SquareCloud) Application runtime environment only Bot deployment and hosting No opt-out
Database Provider Encrypted database storage Persistent data storage No opt-out

All third-party services listed operate under their own privacy policies and are used solely for the technical operation of the Bot.

07

Data Retention

Data is retained for as long as a user's account is active within the Bot's economy system. Specifically:

  • Active accounts: Data is retained indefinitely while the user interacts with the Bot
  • Inactive accounts: Data may be purged after an extended period of inactivity at the administrator's discretion
  • Deletion requests: Users may request the deletion of their economy data by contacting server administration
  • Server removal: If the Bot leaves or is removed from a server, associated data may be retained for a reasonable transition period
08

Your Rights

Depending on your jurisdiction, you may have certain rights regarding your data. Regardless of applicable law, we commit to:

  • Right to access: You can request a summary of the data stored about your account
  • Right to deletion: You can request that your economy data be permanently deleted
  • Right to correction: You can report incorrect data and request corrections through server administration
  • Right to information: You can ask what data is stored about you at any time

To exercise any of these rights, please contact server administration through the authorized Discord server.

09

Children's Privacy

The Bot is not directed at children under the age of 13. We do not knowingly collect data from users under 13 years of age. As Discord itself requires users to be at least 13 years old, any user interacting with the Bot is presumed to meet this minimum age requirement.

If you believe a minor's data has been collected, please contact server administration immediately for removal.

10

Changes to This Privacy Policy

We reserve the right to update this Privacy Policy at any time. When changes are made:

  • The "Effective Date" at the top of this page will be updated
  • Significant changes may be announced within the authorized Discord server
  • Continued use of the Bot after changes are posted constitutes acceptance of the updated policy
11

Contact Us

For any privacy-related questions, data requests, or concerns regarding this Privacy Policy, please contact the server administration directly through the official authorized Discord server.

We are committed to responding to all legitimate privacy inquiries in a timely and transparent manner.
